# 沃草開發種子

# 特色

1. 支援 API 端點
2. 傳統 Flux 架構
3. 使用 React-router
4. API 伺服器自動重啟 [開發模式]
5. 支援 HMR  [開發模式]
6. 基本使用者認證 (TODO)

# 使用方式

1. 複製此資料夾
2. `$ npm install`
3. `$ npm run start-dev`

# 目錄結構

1. `app` 是 react app
    * 傳統 Flux 架構
    * 進入點為 `app/app.es6`

2. `server` 是 API 端點
    * 在 `server/api` 下加入檔案，有三種方式可以新增 API 路由。

        a. 根據檔案名稱: `news.es6` 就會是 `/news`

        b. 根據 Export path 變數:

            const path = "/news/id"
            export default {path}
        c. 根據自定義 path:

            {
                method: "GET",
                path: "/news/my/path",
                handle: function () {}
            }
